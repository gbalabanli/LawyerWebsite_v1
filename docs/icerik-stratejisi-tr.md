## Sayfa İçi SEO Kontrol Listesi (Her Yazı)

- [ ] Başlıkta hedef anahtar kelime geçiyor
- [ ] Açıklama 140-170 karakter, anahtar kelime dahil
- [ ] H2 başlıklarında LSI/ilişkili terimler kullanılmış
- [ ] 1-2 diğer blog yazısına iç bağlantı var
- [ ] WhatsApp danışmanlık CTA'sı var
- [ ] Slug hedef anahtar kelimeye uygun
- [ ] Etiketler: kümeden 2-3 ilgili etiket
